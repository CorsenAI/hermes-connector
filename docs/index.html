Hermes Connector

Attach an exact local Hermes profile and session to only the Chrome tabs you choose. Run simultaneous projects without crossing their browser scopes.

Download companion 0.2.0 View source Setup & support

Explicit scope

Hermes can read or control only tabs you deliberately attach to the selected session. Unbound sessions fail closed.

Local transport

The extension and companion communicate over an authenticated loopback connection. Corsen AI operates no relay or telemetry endpoint.

Your Chrome profile

Use your normal signed-in Chrome profile and its existing site sessions—no hidden automation profile.

Install the companion

Extract the companion ZIP, then run .\install.ps1 on Windows or ./install.sh on macOS/Linux. Restart Hermes, open the extension side panel, and paste the pairing code printed locally by the installer.
