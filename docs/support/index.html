Hermes Connector support

Current public version: 0.2.0 · protocol 3

Install

  1. Download the 0.2.0 companion and extract it.
  2. Run .\install.ps1 on Windows or ./install.sh on macOS/Linux.
  3. Restart any running Hermes dashboard, gateway, or chat process.
  4. Open the Chrome side panel, paste the private pairing code printed locally, select a Hermes session, and attach only the tabs it may control.

Troubleshooting

  • Companion offline: confirm Hermes restarted after installation and port 127.0.0.1:8765 is available.
  • Pairing rejected: open companion settings and generate/show the current local code, then save it again in the extension.
  • Action refused: select the exact Hermes session and attach a permitted web tab. Restricted Chrome pages cannot be controlled.
  • Debugging banner: disable Trusted input unless native input or dialog handling is needed.

Contact

Report reproducible defects in the public issue tracker. For privacy or security questions, email hello@corsen.ai.
